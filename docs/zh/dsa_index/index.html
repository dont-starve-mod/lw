延迟消除工具¶

【独行长路】(Don't Starve Alone) 是一个消除延迟的辅助mod，该mod只适用于单人联机。

开启后，饥荒将按照玩家所处的位置，只运行地面或洞穴一个世界，从而解决延迟问题。

本mod尚处于测试阶段，使用前请仔细阅读下方说明！

启动条件¶

1. 开启本mod¶

这是一个服务器mod，在启动世界前，你需要在服务器mod列表中勾选它。

2. 最大人数为1¶

将世界人数设置为1。 如果人数大于1，服务器仍然可以启动，但本mod不会生效，延迟依旧存在。

3. 世界带有洞穴¶

不含洞穴的世界本就不存在延迟问题。当然，你可以把mod开启，但它不会有任何实际作用。

注意事项¶

1. 存档格式¶

开启【独行长路】的世界，存档格式和饥荒联机官方格式有所不同，所以会存在一些兼容性问题。

官方转mod：

如果你有一个活了100天的普通存档，并且是单人游戏，不需要其他好友加入，则可以直接开启【独行长路】，存档格式将被mod自动转换，完全兼容。

mod转官方：

如果一个世界开启过【独行长路】，那就不要关闭这个mod，因为官方代码并不能识别mod存档文件的洞穴部分，一旦关闭mod，你会丢失在洞穴的所有游戏进度。

换言之，在无法确保一个存档始终不需要其他人进来一起玩的情况下，尽量不要使用本mod。

本mod只适合从开局到结档都是一个人自己玩的存档。

2. 回档¶

当前版本的mod还不支持回档，后续更新会加入。

3. 自动保存¶

开启【独行长路】后，每次下洞穴或者回到地面都会触发一次自动保存。
