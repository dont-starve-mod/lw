人物

出于个人精力原因，我不太想维护这个mod了。

如果你想接手，可以在此mod基础上随意修改或扩展，也可以自己做一个全新的大户爱mod。总之，请自由发挥～

订阅mod¶

  • 联机版

角色介绍¶

大户爱是原创动画《奇蛋物语》（ワンダーエッグ・プライオリティ / Wonder Egg Priority）中的女主角。


基础面板¶

饥饿 200 / 生命 150 / 理智 200 / 攻击系数 1

俗称一威。

梦境强化¶

爱在睡觉可以回复更多生命和理智，但是消耗食物的速度也会变快。

武器¶

大户爱可以制造三种专属武器，武器可以给其他玩家使用。在进入饥荒世界时她自带四色圆珠笔。

四色圆珠笔 Ballpoint pen¶

一支巨大的圆珠笔，挥动它得用点力气，要是被砸中了可不是闹着玩的。

  • 合成公式: 羽毛笔x1，玻璃珠x1
  • 速度: -10%
  • 耐久: 100
  • 伤害: 250
  • 攻击动作时长: 0.66s（前摇）+ 0.42s（后摇）
技巧
  • 圆珠笔爆发伤害很高，但持续输出能力一般。
  • 手动卸下武器可以取消攻击后摇。

荧光棒 Penlight¶

能照明也能砍怪，发出的光芒越强烈，伤害就越高。使用萤火虫和荧光果充能。

  • 合成公式: 提灯x1，电羊角x1
  • 照明时间: 500秒
  • 伤害: 15（熄灭），45-75（点亮）

金色缎带 Golden ribbon¶

甩动时如金蛇狂舞，可以对前方大范围的多个目标造成伤害。

  • 合成公式: 鞭子x1，蜘蛛丝x5
  • 耐久: 500（命中目标数量越多，耐久损失越快）
  • 伤害: 40
  • 可以使用缝纫包和胶布修复。

宠物莱昂（并没有做好）¶

一只能帮忙打架的变色龙。

右键点击莱昂，可以让它伸出舌头把你拉过去，拉动过程无视地形障碍。

注意：爱无法领养其他宠物。
